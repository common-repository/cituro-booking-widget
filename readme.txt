Integrate cituro booking widget to any WordPress website.


== Description ==
The **cituro Booking Widget** plugin adds the [cituro](https://www.cituro.com) booking widget seamlessly to your WordPress website.

It allows visitors to easily make bookings from anywhere on your website through cituro.

The plugin offers extensive customization options, to use the booking widget according to your individual needs.


= What is cituro? =
cituro is a powerful yet easy to use multi-channel online booking system, that makes appointment scheduling effortless for the provider on the one hand and comfortable for the customer on the other hand. cituro also helps to integrate and automate related processes to make them more efficient. For more information visit [www.cituro.com](https://www.cituro.com).

== Installation ==
Simply install and activate the plugin and enter your cituro account number in the plugin settings.
1. Install the plugin directly from WordPress or upload the plugin files to your `\wp-content\plugins\` directory.
2. Activate the plugin in the "Plugins" menu.
3. Navigate to the cituro Booking Widget settings in the WordPress admin menu to configure the plugin.
4. Enter your cituro account number and configure optional customizations.
5. Done. The cituro Booking Widget will show up on your site to receive online bookings


= Requirements =
You need a cituro account to use this plugin. If you don't have an account yet, visit [www.cituro.com](https://www.cituro.com/registrierung) to sign up.


= Use of External Services =
This plugin integrates cituro's booking widget into your website. The required JavaScript will be loaded from cituro servers located at cituro.com or associated subdomains.
